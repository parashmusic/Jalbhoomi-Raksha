Dual Verification Engine

A
Ground Photo Analysis
  • YOLOv8 flood classifier — detects water presence, waterline, submerged structures
  • Depth estimation — monocular depth cues from reference objects (doors, poles)
  • Scene validation — is it really a field/house? Not a stock photo or old image?
  • Metadata hash check — GPS coords match submitted village polygon ± 500m
  • Timestamp freshness — photo within 72h of reported flood event date
✓ Output: Ground Score 0–50 pts
B
Satellite Cross-Validation
  • Sentinel-1 SAR — cloud-free flood mapping via backscatter change (VV/VH ratio)
  • NDWI change detection — pre-event vs post-event water index diff on Sentinel-2
  • SegFormer segmentation — pixel-wise flood extent within village boundary polygon
  • MODIS NDVI loss — crop health degradation confirms agricultural damage
  • Flood % score — what % of claimed area is actually inundated in satellite data
✓ Output: Satellite Score 0–50 pts
C
Score Fusion Logic
  • Total score = Ground Score + Satellite Score (max 100)
  • ≥ 75 → Auto-approved, compensation calculated, DBT queued
  • 55–74 → Approved with officer spot-check within 48h
  • 35–54 → Flagged for field verification by patwari/tehsildar
  • < 35 → Rejected with reason code sent via SMS to Gaon Bura
⚡ Score drives SLA — no human delay for high-confidence claims
D
Anti-Fraud Checks
  • Reverse image search — detect if photo is recycled from web or past claims
  • Duplicate GPS detection — same coordinates claimed by multiple Gaon Buras
  • Historical pattern — village with unusually high claim rate flagged for audit
  • Temporal anomaly — claim submitted before flood event in satellite record
  • Beneficiary dedup — same Aadhaar cannot receive multiple crop loss payments
🔍 Fraud score logged; high-risk claims routed to vigilance

Anti-Fraud Architecture

🚫
Photo Fraud Prevention
Perceptual hash of every uploaded image stored. Future duplicate submissions from any village detected instantly. Stock photos / Google Images detected via reverse search API. EXIF metadata verified — edited images rejected.
📍
Location Spoofing Detection
App uses both GPS and cell tower triangulation. Coordinates must fall within registered village boundary polygon (from Census shapefiles). VPN/mock GPS app detected via Android SafetyNet attestation.
🧾
Beneficiary Deduplication
Each Aadhaar can claim once per disaster event per season. Cross-checked against PM-Fasal Bima Yojana and state relief registers. Linked family members flagged if total claim exceeds household land records.
📊
Statistical Anomaly Detection
ML model trained on historical claim patterns. Village-level Z-score: if claims exceed 3σ above satellite-inferred damage area, auto-flag for audit. Gaon Buras with repeated fraud flags get suspended.
🔗
Blockchain Audit Trail
Every verification event, approval, and payment hashed to a permissioned ledger (Hyperledger Fabric). Immutable record prevents retrospective tampering. CAG and RTI requests answered from ledger.
⏰
Temporal Consistency
Satellite data timestamps are ground truth. Claims submitted for dates where SAR shows no inundation are automatically rejected. Prevents claims for historical floods being resubmitted in new event windows.

# flood_detector.py — Ground photo flood classification using YOLOv8
from ultralytics import YOLO
import torch, cv2, numpy as np
from PIL import Image
from pathlib import Path
from dataclasses import dataclass
from typing import List, Optional
import hashlib, exifread
from datetime import datetime, timedelta

@dataclass
class GroundAnalysisResult:
    flood_detected: bool
    confidence: float       # 0.0 – 1.0
    ground_score: int        # 0 – 50
    damage_class: str        # flood/partial/none
    gps_valid: bool
    timestamp_valid: bool
    is_duplicate: bool
    flags: List[str]

class FloodGroundDetector:
    """YOLOv8-based flood detection from Gaon Bura phone photos"""

    CLASSES = {0: 'flood', 1: 'partial_flood',
               2: 'crop_damage', 3: 'house_damage', 4: 'no_damage'}
    SCORE_MAP = {'flood': 45, 'partial_flood': 30,
                 'crop_damage': 25, 'house_damage': 20, 'no_damage': 5}

    def __init__(self, model_path: str, image_hash_db):
        self.model = YOLO(model_path)
        self.hash_db = image_hash_db  # Redis set of seen hashes

    def analyze(self, image_path: str,
                  submitted_lat: float, submitted_lon: float,
                  event_date: datetime,
                  village_polygon) -> GroundAnalysisResult:

        flags = []
        img = Image.open(image_path)

        # ── 1. Duplicate detection via perceptual hash ─────────────
        img_hash = self._phash(img)
        is_dup = img_hash in self.hash_db
        if is_dup:
            flags.append('DUPLICATE_IMAGE')

        # ── 2. EXIF metadata validation ────────────────────────────
        gps_ok, ts_ok = self._validate_exif(
            image_path, submitted_lat, submitted_lon,
            event_date, village_polygon, flags
        )

        # ── 3. YOLOv8 inference ────────────────────────────────────
        results = self.model.predict(image_path, conf=0.4, verbose=False)
        top_class, confidence = self._parse_preds(results[0])

        # ── 4. Score computation ───────────────────────────────────
        base_score = self.SCORE_MAP.get(top_class, 0)
        score = base_score
        if not gps_ok:   score -= 15; flags.append('GPS_MISMATCH')
        if not ts_ok:    score -= 10; flags.append('TIMESTAMP_STALE')
        if is_dup:       score  = 0

        # Store hash to prevent reuse
        self.hash_db.add(img_hash)

        return GroundAnalysisResult(
            flood_detected=(top_class in ['flood', 'partial_flood']),
            confidence=confidence,
            ground_score=max(0, min(50, score)),
            damage_class=top_class,
            gps_valid=gps_ok,
            timestamp_valid=ts_ok,
            is_duplicate=is_dup,
            flags=flags
        )

    def _validate_exif(self, path, lat, lon, event_dt, polygon, flags):
        with open(path, 'rb') as f:
            tags = exifread.process_file(f, stop_tag='GPS')

        # GPS coords from EXIF must be within village polygon
        exif_lat = self._dms_to_dd(tags.get('GPS GPSLatitude'))
        exif_lon = self._dms_to_dd(tags.get('GPS GPSLongitude'))
        gps_ok = polygon.buffer(0.005).contains(  # ~500m tolerance
            Point(exif_lon, exif_lat)) if exif_lat else False

        # Photo must be taken within 72h of event
        exif_dt_str = str(tags.get('EXIF DateTimeOriginal', ''))
        try:
            exif_dt = datetime.strptime(exif_dt_str, '%Y:%m:%d %H:%M:%S')
            ts_ok = abs((exif_dt - event_dt).total_seconds()) < 72 * 3600
        except:
            ts_ok = False

        return gps_ok, ts_ok

    def _phash(self, img) -> str:
        img_small = img.convert('L').resize((16, 16))
        pixels = list(img_small.getdata())
        avg = sum(pixels) / len(pixels)
        bits = ''.join('1' if p > avg else '0' for p in pixels)
        return hex(int(bits, 2))

# sar_processor.py — Sentinel-1 SAR flood extent mapping
import ee
import numpy as np
import geopandas as gpd
from shapely.geometry import shape, mapping
from dataclasses import dataclass

@dataclass
class SatelliteResult:
    flooded_area_ha: float
    flood_percentage: float   # % of claimed area inundated
    satellite_score: int      # 0 – 50
    ndvi_loss: float          # crop health change
    confidence: str           # HIGH / MEDIUM / LOW

class SARFloodMapper:
    """Sentinel-1 SAR based flood mapping via GEE. Works through monsoon clouds."""

    def __init__(self):
        ee.Initialize()

    def analyze_village(self, village_geojson: dict,
                           event_date: str,      # 'YYYY-MM-DD'
                           claimed_area_ha: float) -> SatelliteResult:

        aoi = ee.Geometry.Polygon(
            village_geojson['coordinates'])

        pre_start  = self._offset_date(event_date, -30)
        pre_end    = self._offset_date(event_date, -3)
        post_end   = self._offset_date(event_date, 7)

        # ── Sentinel-1 SAR collection ──────────────────────────────
        s1_pre = (ee.ImageCollection('COPERNICUS/S1_GRD')
                    .filterBounds(aoi)
                    .filterDate(pre_start, pre_end)
                    .filter(ee.Filter.eq('instrumentMode', 'IW'))
                    .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VV'))
                    .select('VV').mean())

        s1_post = (ee.ImageCollection('COPERNICUS/S1_GRD')
                     .filterBounds(aoi)
                     .filterDate(event_date, post_end)
                     .filter(ee.Filter.eq('instrumentMode', 'IW'))
                     .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VV'))
                     .select('VV').mean())

        # ── Flood mask: backscatter drops during flooding ──────────
        # Water absorbs radar → VV drops significantly
        diff = s1_pre.subtract(s1_post)
        flood_mask = diff.gt(3)  # 3 dB threshold (empirically validated)

        # ── Compute flooded area ───────────────────────────────────
        pixel_area = ee.Image.pixelArea().divide(10000)  # m² → ha
        flooded_ha = (flood_mask.multiply(pixel_area)
                               .reduceRegion(
                                   reducer=ee.Reducer.sum(),
                                   geometry=aoi, scale=10)
                               .getInfo()).get('VV', 0)

        # ── NDVI loss for crop damage ──────────────────────────────
        ndvi_loss = self._compute_ndvi_loss(aoi, pre_end, post_end)

        # ── Scoring ────────────────────────────────────────────────
        flood_pct = min(1.0, flooded_ha / claimed_area_ha) if claimed_area_ha > 0 else 0
        sat_score = int(flood_pct * 50)
        confidence = 'HIGH' if flood_pct > 0.7 else 'MEDIUM' if flood_pct > 0.3 else 'LOW'

        return SatelliteResult(
            flooded_area_ha=round(flooded_ha, 2),
            flood_percentage=round(flood_pct * 100, 1),
            satellite_score=sat_score,
            ndvi_loss=ndvi_loss,
            confidence=confidence
        )

    def _compute_ndvi_loss(self, aoi, pre_end, post_end) -> float:
        def get_ndvi(date_range):
            return (ee.ImageCollection('COPERNICUS/S2_SR_HARMONIZED')
                      .filterBounds(aoi).filterDate(*date_range)
                      .filter(ee.Filter.lt('CLOUDY_PIXEL_PERCENTAGE', 30))
                      .median().normalizedDifference(['B8', 'B4'])
                      .reduceRegion(ee.Reducer.mean(), aoi, 20)
                      .getInfo().get('nd', 0))

        pre_ndvi  = get_ndvi([self._offset_date(pre_end, -20), pre_end])
        post_ndvi = get_ndvi([self._offset_date(post_end, -10), post_end])
        return round(pre_ndvi - post_ndvi, 3)

    def _offset_date(self, date_str: str, days: int) -> str:
        from datetime import datetime, timedelta
        dt = datetime.strptime(date_str, '%Y-%m-%d') + timedelta(days=days)
        return dt.strftime('%Y-%m-%d')

# verification_engine.py — Score fusion + claim routing
from dataclasses import dataclass
from enum import Enum
from flood_detector import FloodGroundDetector, GroundAnalysisResult
from sar_processor import SARFloodMapper, SatelliteResult
import logging

class ClaimStatus(Enum):
    AUTO_APPROVED      = "AUTO_APPROVED"      # Score ≥ 75
    OFFICER_REVIEW     = "OFFICER_REVIEW"     # Score 55–74
    FIELD_VERIFICATION = "FIELD_VERIFICATION" # Score 35–54
    REJECTED           = "REJECTED"           # Score < 35
    FRAUD_FLAGGED      = "FRAUD_FLAGGED"      # Fraud detected

@dataclass
class VerificationResult:
    claim_id: str
    total_score: int           # 0 – 100
    ground_score: int
    satellite_score: int
    status: ClaimStatus
    estimated_compensation: float  # INR
    flood_area_ha: float
    flags: list
    sms_message: str

class VerificationEngine:
    def __init__(self, ground_detector, sar_mapper, compensation_calc):
        self.ground   = ground_detector
        self.sar      = sar_mapper
        self.comp_calc = compensation_calc

    def process_claim(self, claim: dict) -> VerificationResult:
        logging.info(f"Processing claim {claim['claim_id']}")

        # ── Step 1: Analyze ground photos ─────────────────────────
        ground_results = []
        for photo_path in claim['photo_paths']:
            r = self.ground.analyze(
                photo_path,
                claim['submitted_lat'], claim['submitted_lon'],
                claim['event_date'], claim['village_polygon']
            )
            ground_results.append(r)

        # Take best ground score from multi-photo submission
        best_ground = max(ground_results, key=lambda x: x.ground_score)
        all_flags   = [f for r in ground_results for f in r.flags]

        # ── Step 2: Satellite cross-validation ────────────────────
        sat_result = self.sar.analyze_village(
            claim['village_geojson'],
            claim['event_date'],
            claim['claimed_area_ha']
        )

        # ── Step 3: Fraud check override ──────────────────────────
        if 'DUPLICATE_IMAGE' in all_flags:
            return self._build_result(claim['claim_id'], 0, 0, 0,
                ClaimStatus.FRAUD_FLAGGED, 0, 0, all_flags,
                "⚠ Claim rejected: duplicate image detected. Ref: FRAUD-PHOTO")

        # ── Step 4: Compute total score ───────────────────────────
        total = best_ground.ground_score + sat_result.satellite_score

        # ── Step 5: Route by score ────────────────────────────────
        if   total >= 75: status = ClaimStatus.AUTO_APPROVED
        elif total >= 55: status = ClaimStatus.OFFICER_REVIEW
        elif total >= 35: status = ClaimStatus.FIELD_VERIFICATION
        else:             status = ClaimStatus.REJECTED

        # ── Step 6: Calculate compensation ────────────────────────
        comp = self.comp_calc.calculate(
            area_ha=sat_result.flooded_area_ha,
            crop_type=claim['crop_type'],
            damage_pct=sat_result.flood_percentage / 100,
            state=claim['state']
        ) if status != ClaimStatus.REJECTED else 0

        sms = self._build_sms(claim['claim_id'], total, status, comp)

        return VerificationResult(
            claim_id=claim['claim_id'], total_score=total,
            ground_score=best_ground.ground_score,
            satellite_score=sat_result.satellite_score,
            status=status, estimated_compensation=comp,
            flood_area_ha=sat_result.flooded_area_ha,
            flags=all_flags, sms_message=sms
        )

    def _build_sms(self, cid, score, status, comp) -> str:
        status_map = {
            ClaimStatus.AUTO_APPROVED:      f"✓ APPROVED. Payment ₹{comp:,.0f} in 48h.",
            ClaimStatus.OFFICER_REVIEW:     "Under review by district officer (48h).",
            ClaimStatus.FIELD_VERIFICATION: "Field survey scheduled in 5 days.",
            ClaimStatus.REJECTED:           "Insufficient evidence. Submit via portal.",
        }
        return f"BhumiRaksha [{cid}]: {status_map.get(status, '')} Score:{score}/100"

# compensation.py — NDRF rate-based compensation calculator
from dataclasses import dataclass

# NDRF norms (2023 revision) — INR per hectare
CROP_RATES = {
    'paddy_rainfed':    6800,
    'paddy_irrigated':  13500,
    'wheat':            6800,
    'jute':             6800,
    'sugarcane':        18000,
    'vegetables':       13500,
    'perennial_crops':  18000,
}

HOUSE_RATES = {
    'pucca_full':     95100,
    'pucca_partial':  5200,
    'kutcha_full':    3200,
    'kutcha_partial': 3200,
}

# State-specific multipliers (Assam, Bihar get higher due to frequency)
STATE_MULTIPLIERS = {
    'Assam':       1.2,
    'Bihar':       1.15,
    'Odisha':      1.1,
    'Uttar Pradesh': 1.0,
    'West Bengal': 1.05,
}

@dataclass
class CompensationBreakdown:
    crop_loss: float
    house_damage: float
    total: float
    breakdown_str: str

class CompensationCalculator:
    def calculate(self,
                   area_ha: float,
                   crop_type: str,
                   damage_pct: float,   # 0.0–1.0
                   state: str,
                   house_damage_type: str = None) -> CompensationBreakdown:

        multiplier = STATE_MULTIPLIERS.get(state, 1.0)
        rate = CROP_RATES.get(crop_type, 6800)

        # Crop loss: area × rate × damage% × state multiplier
        crop_loss = area_ha * rate * damage_pct * multiplier

        # House damage (if reported)
        house_loss = HOUSE_RATES.get(house_damage_type, 0) if house_damage_type else 0

        total = crop_loss + house_loss

        breakdown = (
            f"Crop: {area_ha:.1f}ha × ₹{rate} × {damage_pct*100:.0f}% × {multiplier} = ₹{crop_loss:,.0f}\n"
            f"House: ₹{house_loss:,.0f}\nTotal: ₹{total:,.0f}"
        )

        return CompensationBreakdown(
            crop_loss=round(crop_loss, 2),
            house_damage=round(house_loss, 2),
            total=round(total, 2),
            breakdown_str=breakdown
        )

    def bulk_calculate(self, claims_df) -> dict:
        "Process a DataFrame of approved claims, return totals by district"
        results = claims_df.apply(
            lambda row: self.calculate(
                row.flooded_ha, row.crop_type,
                row.damage_pct, row.state
            ).total, axis=1
        )
        return {
            'individual_amounts': results.tolist(),
            'district_totals': claims_df.groupby('district')['claim_id'].count().to_dict(),
            'total_outgo': results.sum()
        }

    # Send SMS to Gaon Bura
    await send_sms(village_id, result.sms_message)

    return ClaimStatusResponse(
        claim_id=claim_id,
        status=result.status.value,
        total_score=result.total_score,
        estimated_compensation=result.estimated_compensation,
        flood_area_ha=result.flood_area_ha,
        message=result.sms_message
    )
